Teste do Header com Botão Sair

Este é um teste para verificar se o botão "Sair" está visível no header.

  • Desktop: O botão "Sair" deve aparecer com o texto "Sair" em vermelho
  • Mobile: O botão "Sair" deve aparecer com o símbolo "✕" em vermelho
  • Funcionalidade: Ao clicar, deve mostrar um alert "Logout clicado!"

Instruções de teste:

  1. Redimensione a janela para testar responsividade
  2. Clique no botão vermelho "Sair" no canto superior direito
  3. Verifique se funciona tanto em desktop quanto em mobile
  4. Depois acesse a aplicação real para testar
